# Tier 02 — Realistic Apps

End-to-end apps riding 2025–2026 developer demand. These samples justify telling someone "try ato" by giving them something they'd actually run.

## Flagships

1. **`byok-chat-openrouter`** — BYOK chat UI where env + `allow_domains` tell the entire security story in 4 lines of config.
2. **`ollama-local-rag`** — Local RAG over PDFs with zero egress. Proves the sandbox is useful offline.
3. **`scraper-with-allowlist`** — Pure fail-closed network demo.
