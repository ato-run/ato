# ato-samples

Runnable samples for [`ato`](https://ato.run) — the Nix alternative for the AI era. Every sample here is guaranteed to run on any user machine when CI is green.

> **Backwards-compatibility contract:** from `ato` v0.5 onward, every sample in this repository runs unchanged against every released `ato` version ≥ its declared `min_ato_version`. CI enforces this on every PR.

## How to read this repo

Four tiers, climbing from "wow" to "real work" to "here's what we honestly can't do":

| Tier | What it is | When to use it |
|---|---|---|
| [`00-quickstart/`](00-quickstart/) | ≤60-second first-run demos, one per runtime | Your first encounter with `ato` |
| [`01-capabilities/`](01-capabilities/) | One sample per `ato` feature (encap/decap, watch, background, ipc, network policy, env preflight) | Learn a specific command or flag |
| [`02-apps/`](02-apps/) | Realistic end-to-end apps (BYOK chat, local RAG, scraper, dashboard) | See how `ato` runs real workloads |
| [`03-limitations/`](03-limitations/) | Runnable "expected to fail" demos | Understand what `ato` does **not** do |

Difficulty legend: `beginner` · `intermediate` · `advanced`.
Time legend: `~1min` · `~5min` · `~15min` · `~30min`.

## Running a sample

```bash
# Install ato (see https://ato.run)
curl -fsSL https://ato.run/install.sh | sh

# Clone this repo
git clone https://github.com/ato-run/ato-samples.git
cd ato-samples

# Run any sample
cd 00-quickstart/url-to-run-hello-static
ato run .
```

Or run directly from a URL without cloning:

```bash
ato run github.com/ato-run/ato-samples/00-quickstart/url-to-run-hello-static
```

## Every sample ships with

- `capsule.toml` — the `ato` manifest (runtime, permissions, commands)
- `health.toml` — CI health-check spec (see [`tools/schemas/health.schema.json`](tools/schemas/health.schema.json))
- `README.md` — YAML front-matter (title, difficulty, time, tags, requires, verified_on, ato_version) + explanation
- `EXPECTED.md` — what a successful (or expected-failure) run looks like
- Committed lockfiles (`capsule.lock.json`, `ato.lock.json`, plus runtime lockfiles: `pnpm-lock.yaml`, `uv.lock`, etc.)
