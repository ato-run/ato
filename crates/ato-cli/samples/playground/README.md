# Playground

Experimental BYOK sample apps. Looser contract than the four curated tiers — no L3+ CI, no backwards-compat guarantee, and no guarantee they pass on every OS.

> Inclusion here does not imply `ato` endorsement. Promote to `02-apps/` if they become polished and demand-worthy.

## Policy

- Must still ship `capsule.toml` and a minimal `health.toml` (at least L0 + L1).
- Breakage on main will open a notice issue but will not block releases.
- Retirement policy: no activity for two release cycles → moved to `SHOWCASE.md` as a historical link, then deleted.
