Desky UI Design Guidelines

Version 2.4 · Last updated 3 min ago by Agent · Approved

Core Principles

Desky follows a Single Window, Multi-Split paradigm. Every interaction happens within one unified canvas, eliminating window clutter while preserving parallel work capability.

  • Central Focus — Main Stage always dominates visual hierarchy
  • Peripheral Retreat — Navigation elements stay dormant until needed
  • Spatial Depth — Task Rail uses filmstrip perspective for spatial memory
  • Translucent Intelligence — Agent UI peeks without obstructing

Command Bar Spec

The top bar integrates title, navigation, and AI input into a single slim region (48px).

/* Command Bar */ .command-bar { height: 48px; background: glass-heavy; backdrop-filter: blur(40px); border-bottom: 1px solid subtle; }

Agent Integration

Agent status via minimal glowing indicator. Detailed activity in right-side Peek Panel.

  • Status Dot indicator — pulse animation
  • Peek Slide-over diagnostics panel
  • Approval Inline approval requests
