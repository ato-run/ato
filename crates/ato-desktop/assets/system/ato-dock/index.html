/ New Capsule

Source Input

Enter a public GitHub repository URL. Private repositories are not supported in this version.

What happens here

Ato clones the repository or reads the local path, checks for a capsule.toml manifest, and prepares the working directory.

  • Public GitHub repos are cloned to ~/.ato/dock/sources/
  • Local paths are used in-place
  • Private repositories are not supported yet

Install & Manifest

Editing the manifest

The manifest declares your capsule's name, description, runtime, and permissions. Save draft to write changes back to capsule.toml.

What is verified

Ato runs ato publish --build --json and checks three phases:

  • Prepare — resolve dependencies
  • Build — compile or bundle source
  • Verify — check capsule integrity
